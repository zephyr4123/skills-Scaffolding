# swiftui-pro

- **来源**：Paul Hudson（Hacking with Swift）— <https://github.com/twostraws/swiftui-agent-skill>
- **安装**：`/plugin marketplace add twostraws/swiftui-agent-skill`，然后 `/plugin install swiftui-pro@swiftui-agent-skill`
- **本机版本**：1.0.0（2026-06-08 装）
- **一句话**：SwiftUI 专业代码审查插件——按九份参考清单系统检查 SwiftUI 项目的现代 API、数据流、导航、无障碍与性能。

## Skill 清单

| Skill | 是干啥的 | 什么时候用 |
|---|---|---|
| swiftui-pro | 依据 api/views/data/导航/设计/无障碍/性能等九份参考文档，全面审查 SwiftUI 代码的最佳实践 | 读、写或审查 SwiftUI 项目代码时，检查现代 API 用法、可维护性与性能 |
